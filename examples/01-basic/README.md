# Basic Examples

Start here! These examples teach the fundamentals of the Tawk Agents SDK.

## Examples

1. **01-simple-agent.ts** - Your first agent (Hello World)
2. **02-agent-with-tools.ts** - Give your agent capabilities  
3. **03-multi-agent.ts** - Multiple agents working together
4. **04-sessions.ts** - Remember conversation history

## Quick Start

```bash
# Run the first example
npx tsx 01-basic/01-simple-agent.ts

# Or use the runner
npx tsx run.ts 01
```
